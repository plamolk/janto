⬅️ กลับไปหน้าหลัก

💡 การใช้งานเบื้องต้น

ระบบนี้ออกแบบมาให้ใช้งานง่ายและปลอดภัย ข้อมูลทุกอย่างที่คุณบันทึกจะถูก บันทึกอัตโนมัติ (Auto-Save) ลงในเครื่องคอมพิวเตอร์ของคุณทันที และจะ สำรองข้อมูลขึ้น Google Drive อัตโนมัติ (Auto-Backup) เมื่อมีการเชื่อมต่ออินเทอร์เน็ต

คุณไม่จำเป็นต้องกดปุ่มสำรองข้อมูลใดๆ ด้วยตัวเอง ระบบจัดการให้คุณทั้งหมด!

🛠️ การแจ้งเตือนและวิธีแก้ไข

1. ⚠️ ไม่มีอินเทอร์เน็ต (No Internet)

  • สาเหตุ: คอมพิวเตอร์ไม่ได้เชื่อมต่อเน็ต ข้อมูลจะบันทึกแค่ในเครื่อง ยังไม่ขึ้น Cloud
  • วิธีแก้: เช็คสาย LAN หรือ Wi-Fi ของร้าน เมื่อเน็ตกลับมา แถบเตือนจะหายไปเอง

2. ⛔ เวลาเครื่องไม่ตรง (Time Desync / Read-Only Mode)

  • สาเหตุ: นาฬิกาคอมพิวเตอร์เดินไม่ตรงกับเวลามาตรฐานสากล ระบบจะล็อคการบันทึกเพื่อป้องกันข้อมูลรวน
  • วิธีแก้: ไปที่ตั้งค่า (Settings) ของ Windows/Mac → Date & Time → เปิด "Set time automatically" (ตั้งเวลาอัตโนมัติ) แล้วกด F5 รีเฟรชหน้าเว็บ

3. ⛔ ไม่พบไฟล์ token.json / Google API Error

  • สาเหตุ: ระบบสูญเสียกุญแจเชื่อมต่อ Google Drive
  • วิธีแก้: 🛠️ กรุณาติดต่อฝ่ายไอที (โทร: 0936108294)

4. ⛔ ฐานข้อมูลเสียหาย (Database Corrupt)

  • สาเหตุ: ไฟล์เก็บข้อมูลเกิดความเสียหาย (อาจเกิดจากไฟตกตอนกำลังเซฟ)
  • วิธีแก้: 🛑 ห้ามใช้งานต่อเด็ดขาด! 🛠️ กรุณาติดต่อฝ่ายไอที (โทร: 0936108294) ทันที

5. ⛔ ไม่สามารถเชื่อมต่อเซิร์ฟเวอร์ได้ (Backend Offline)

  • สาเหตุ: โปรแกรมหลังบ้านปิดตัวลง
  • วิธีแก้: ลองรีสตาร์ทโปรแกรม 1 ครั้ง หากไม่หาย 🛠️ กรุณาติดต่อฝ่ายไอที (โทร: 0936108294)
